Backlight adjustment method

1. Use the WS_xinchDSI_Backlight_arm64 executable file in the Backlight folder
1. CD Backlight
2. sudo chmod 777 WS_xinchDSI_Backlight_arm64
3. Run sudo ./WS_xinchDSI_Backlight_arm64 on the terminal or double-click WS_xinchDSI_Backlight_arm64

2. Install WS_xinchDSI_Backlight_arm64 to Accessories
1. CD Backlight
2. sudo chmod 777 install.sh
3. sudo ./install.sh
4. Open the program in Start Menu->Accessories
